# LifeOS Central

## Project info

This is the LifeOS Central application.

## Getting Started

1. **Clone the repository**:
   ```sh
   git clone <YOUR_GIT_URL>
   cd lifeos-central
   ```

2. **Install dependencies**:
   ```sh
   npm i
   ```

3. **Start the development server**:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
